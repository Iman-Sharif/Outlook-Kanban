Usage

Basics

  • Drag a task card between lanes.
  • Click a card to open the drawer (details, notes, subtasks, quick actions).
  • Double-click a card to open it in Outlook.
  • Keyboard: focus a card with Tab, press Enter to open the drawer, and use Space to select.

Snooze (due date presets)

  • Use the drawer Snooze buttons to set a due date relative to today.
  • You can also use Card Actions snooze presets.
  • For multiple tasks, select them and use the bulk bar Snooze button.

Selection

  • Select tasks using the checkbox on cards.
  • Shift+click selects a range within the same lane.
  • Use the header Select menu for bulk selection actions.

Filters

  • Search filters by subject and notes preview.
  • Filter by category, due, status, stale, and privacy.
  • On narrow panes, filters collapse into a single Filters button.

Notes and subtasks

  • Notes are stored in the Outlook task Body.
  • Subtasks are stored as Markdown-style checkboxes in the Body.

Tip: open kanban.html inside Outlook (Folder Home Page) to use the board.

Accessibility

Kanban for Outlook runs inside classic Outlook's Folder Home Page (legacy IE engine). Some behaviours depend on the host.

Keyboard

  • Use Tab to focus a task card.
  • Press Enter or Space to open the task drawer.
  • Esc closes popovers, dialogs, and the drawer.

Dragging alternatives

  • Move between lanes: use Card Actions -> "Move to lane..." (or bulk actions).
  • Manual ordering (when enabled): use Card Actions to move up/down/top/bottom.

Focus

  • Popovers and dialogs keep focus inside while open.
  • When a popover/dialog closes, focus returns to the control that opened it (best-effort).

Setup

Requirements

  • Windows + classic Outlook for Windows (not the new Outlook)
  • Folder Home Pages must be allowed by your organisation (some policies disable them)

Install / upgrade (recommended)

  1. Extract the release zip (it contains a kanban-for-outlook folder).
  2. Open that folder and run install.cmd.
  3. Choose Install / Upgrade from the menu.
  4. Restart Outlook.

Windows SmartScreen

If Windows shows "Windows protected your PC", click "More info" then "Run anyway".

Manual setup (no scripts)

  1. Copy the extracted folder to a location you control.
  2. In Outlook: right-click the folder you want the board on (often your mailbox root) then choose Properties.
  3. Home Page tab: set the Address to your local kanban.html and enable "Show home page by default for this folder".

Uninstall

Run install.cmd and choose Uninstall.

Troubleshooting

  • If your organisation blocks Folder Home Pages, the app cannot run.
  • If you opened kanban.html in a normal browser, it will show the "Home Page" warning. It must run inside Outlook.
  • If you are using the new Outlook or Outlook on the web: they are not supported.

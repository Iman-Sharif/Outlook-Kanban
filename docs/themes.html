Themes

Built-in themes

Use Settings -> Appearance to switch between the built-in Light and Dark themes.

Custom themes (local CSS)

  • You can import a local .css file as a theme from Settings -> Appearance.
  • The app blocks remote loads and scriptable CSS. Themes must be local-only.

Theme management

  • Apply: switch to a theme.
  • Rename/remove: tidy up imported themes you no longer need.
  • Reset: Settings has per-section reset buttons if you want to return to defaults.

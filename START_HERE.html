Kanban for Outlook - Start here

This app runs locally inside classic Outlook for Windows (Folder Home Page).

Install (recommended)

  • Extract the downloaded zip.
  • Run install.cmd.
  • Restart Outlook.

If Windows SmartScreen blocks the script: click "More info" -> "Run anyway" (or use manual install).

Did you use "Code -> Download ZIP"?

That zip is the source code (developer download) and contains extra files.

You can still install from it by running install.cmd, but the easiest path is to download the official release zip.

Download links

Download latest release zip

On the Releases page, ignore the auto-generated Source code (zip/tar.gz) downloads. You want kanban-for-outlook.zip.

Install options

  • Recommended: run install.cmd (copies files to your profile and registers the home page).
  • Manual: follow the steps in docs/SETUP.md (no scripts required).

Troubleshooting

  • If your organisation blocks Folder Home Pages, the app cannot run. See docs/SETUP.md (Troubleshooting section).
  • If you are using the new Outlook: it is not supported.

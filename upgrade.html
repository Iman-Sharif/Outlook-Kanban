Upgrade (Local Installation)

This application is designed to run fully locally. Upgrading is a file replacement.

  1. Close Outlook.
  2. Extract the new release zip to a temporary folder.
  3. Copy the extracted files over your install folder.
  4. Run install.cmd again (safe to re-run) and choose Install / Upgrade.
  5. Start Outlook.

Tip: Your configuration is stored locally in Outlook (Journal item) and will be preserved.

Note: If you are upgrading from an older JanBan-based installation, this fork may create a new configuration format. Your previous configuration is backed up locally in Outlook as a Journal item with a subject like KanbanConfig.legacy.YYYYMMDD-HHMMSS.

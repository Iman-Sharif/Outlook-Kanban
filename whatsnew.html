What's New

This page is local-only and ships with the app.

v3.0.0

  • Local-only hardening (no external downloads, update checks, or support email targets).
  • Projects as Outlook Task folders.
  • Custom lanes stored on tasks (drag-and-drop updates a local Outlook user property).
  • Migration tool to bulk-assign lanes from existing Outlook Task Status values.
  • Project tools: link existing folders, hide/show projects, rename projects, move tasks between projects.
  • Modern light/dark themes and theme import/folder-theme support.
  • More customization: density, motion, lane width, card field toggles, and status sync controls.

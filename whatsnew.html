What's New

This page is local-only and ships with the app.

v1.3.0

  • Snooze quick actions (drawer buttons + card Actions presets), computed from today.
  • Keyboard access improvements: focus task cards with Tab and open the drawer with Enter/Space.
  • Better focus handling for popovers and dialogs (focus trap + return focus).
  • Manual ordering without drag-and-drop (Actions: move up/down/top/bottom).

v1.2.0

  • Select menu plus Shift+click range selection (within a lane).
  • Responsive Filters button/menu with an active-count badge.
  • Card indicators for checklist progress and notes (Comfortable mode).
  • Optional: highlight search matches in card titles and notes.
  • Settings search, per-section reset, and theme management (apply/rename/remove).
  • Drawer field buttons to change lane, due, categories, priority, and privacy.

v1.1.0

  • Edit task notes directly in the task drawer (with Undo).
  • Checklist support in the drawer using Markdown-style checkboxes (- [ ]).
  • Clearer colour cues for task actions (green Complete, red Delete).

v1.0.0

  • Initial release of this repository.
  • Local-only Kanban board for classic Outlook Tasks (Folder Home Page / legacy IE engine).
  • Projects (folders), configurable lanes stored on tasks, drag-and-drop with Undo, and powerful filters.
  • Views, tools (migration + move between projects), and settings export/import.
  • Light/Dark mode, Compact mode, and a safer, friendlier Settings experience.

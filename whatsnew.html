What's New

This page is local-only and ships with the app.

v3.5.4

  • Added a Due filter and clearer filter chips.
  • Added Undo for lane moves and an optional drag handle to reduce accidental drags.
  • Improved WIP feedback, refresh UX, and project selection.
  • Added a safe lane id rename tool (migrates tasks, then updates lane config).

v3.5.3

  • New task entrypoint now uses quick add (current folder) with improved empty states.
  • Better filter feedback and safer drag/drop behaviour while filtered.
  • Lanes editor redesigned for readability, with improved safety and dark mode polish.
  • Improved behaviour in classic Outlook/IE11 (scrolling and focus rings).

v3.5.2

  • Projects are now optional: the board always includes your default Outlook Tasks folder.
  • Built-in themes renamed to Light mode / Dark mode, with Dark mode tuned to match Outlook colours.
  • Settings switched to a single-column layout and repeated disclaimer messaging reduced.

v3.5.1

  • Clearer download and setup guidance for non-technical users.
  • Added a local Start Here page in the release zip.

v3.5.0

  • Releases are now published via GitHub Releases with a SHA-256 checksum for verification.

v3.4.0

  • Hardened theme safety checks and local-only guardrails.
  • Theme validation now runs when applying a theme (defence-in-depth).

v3.3.0

  • Export/import settings (config + view state) as a local JSON file.
  • Opt-in keyboard shortcuts (designed to avoid Outlook key conflicts).
  • Quick add tasks into a lane without opening the Outlook inspector.

v3.2.0

  • Maintainability refactor: split the controller into ES5 modules.
  • Outlook adapter boundary: handle COM exceptions in one place.
  • Node-only unit tests for pure logic.

v3.1.0

  • Expanded Diagnostics (including storage health and refresh timings).
  • Consistent local error UX (toasts + error details modal).
  • Clearer messaging when Folder Home Pages are blocked by policy.

v3.0.0

  • Local-only hardening (no external downloads, update checks, or support email targets).
  • Projects as Outlook Task folders.
  • Custom lanes stored on tasks (drag-and-drop updates a local Outlook user property).
  • Migration tool to bulk-assign lanes from existing Outlook Task Status values.
  • Project tools: link existing folders, hide/show projects, rename projects, move tasks between projects.
  • Modern light/dark themes and theme import/folder-theme support.
  • More customisation: density, motion, lane width, card field toggles, and status sync controls.

See upgrade.html for upgrade instructions.
